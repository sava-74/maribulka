# Инструкция по запуску MCP-серверов

Этот документ содержит инструкции по запуску MCP-серверов, включая основной сервер basic-memory.

## MCP Серверы для проекта Maribulka

## 1. Назначение и возможности серверов

Sequential-thinking сервер предоставляет цепочку мыслей и логический анализ. Позволяет проводить сложные рассуждения пошагово, поддерживает логические выводы и анализ ситуаций, помогает разбивать сложные проблемы на подзадачи и строить логические цепочки рассуждений.

Context7-mcp сервер обеспечивает управление контекстом и долгосрочной памятью. Хранит информацию между сессиями, управляет памятью для ИИ-ассистентов, позволяет сохранять и извлекать данные из документации и справочных материалов.

Playwright-mcp сервер обеспечивает автоматизацию браузера и веб-тестирование. Управляет браузерами, выполняет действия на веб-страницах, автоматизирует тестирование UI, позволяет навигироваться по сайтам, заполнять формы, кликать по элементам и оценивать результаты действий.

Basic-memory сервер обеспечивает базовое хранение и извлечение информации. Обеспечивает простое хранение данных, извлечение информации, поддержку краткосрочной памяти, позволяет сохранять и извлекать фрагменты информации и управлять контекстом разговора или сессии.

## 2. Запуск серверов

**Sequential-thinking сервер:**

- Для запуска выполните: `npx -y @modelcontextprotocol/server-sequential-thinking`

**Context7-mcp сервер:**

- Для запуска выполните: `npx -y @upstash/context7-mcp@latest`

**Playwright-mcp сервер:**

- Для запуска выполните: `npx @playwright/mcp@latest`

**Basic-memory сервер:**

- Для запуска выполните: `uvx basic-memory mcp`

## 3. Инструменты серверов

**Sequential-thinking** сервер:

- Инструмент **sequentialthinking** - позволяет проводить пошаговый анализ задач, разбивать сложные проблемы на подзадачи, строить логические цепочки рассуждений

**Context7-mcp** сервер:

- Инструмент **query-docs** - позволяет делать запросы к документации библиотек, находить нужную информацию в справочных материалах
- Инструмент **resolve-library-id** - позволяет определять идентификаторы библиотек для использования в запросах

**Playwright-mcp** сервер:

- Инструмент **navigate** - позволяет переходить по URL в браузере
- Инструмент **snapshot** - позволяет захватывать состояние страницы
- Инструмент **click** - позволяет кликать по элементам на странице
- Инструмент **fill_form** - позволяет заполнять формы
- Инструмент **evaluate** - позволяет выполнять JavaScript на странице
- Инструмент **wait_for** - позволяет ожидать появления или исчезновения текста или задержки времени
- Инструмент **type** - позволяет вводить текст в поля ввода
- Инструмент **select_option** - позволяет выбирать опции в выпадающих списках
- Инструмент **press_key** - позволяет нажимать клавиши на клавиатуре
- Инструмент **network_requests** - позволяет просматривать сетевые запросы
- Инструмент **console_messages** - позволяет просматривать сообщения консоли
- Инструмент **tabs** - позволяет управлять вкладками браузера
- Инструмент **resize** - позволяет изменять размер окна браузера
- Инструмент **take_screenshot** - позволяет делать скриншоты
- Инструмент **handle_dialog** - позволяет обрабатывать диалоговые окна
- Инструмент **hover** - позволяет наводить курсор на элементы
- Инструмент **file_upload** - позволяет загружать файлы
- Инструмент **run_code** - позволяет выполнять произвольный код Playwright
- Инструмент **close** - позволяет закрыть страницу
- Инструмент **drag** - позволяет перетаскивать элементы

**Basic-memory** сервер:

- Инструмент **memory** - позволяет сохранять и извлекать фрагменты информации
- Инструмент **context** - позволяет управлять контекстом разговора или сессии
